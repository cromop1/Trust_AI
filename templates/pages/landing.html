Puesta en marcha express

1

Instala requisitos

Ejecuta pip install -r requirements.txt para obtener Django, Pillow y Requests.

2

Configura la clave

Define la variable DEEPSEEK_API_KEY en tu entorno o archivo .env.

3

Aplica migraciones

Ejecuta python manage.py migrate y crea tu usuario administrador.

4

Lanza el servidor

Arranca con python manage.py runserver y accede a /app/ para usar el panel.
